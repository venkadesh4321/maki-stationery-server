## Initial Setup

1. Install dependencies
```bash
npm install
```

2. Configure environment
```bash
cp .env.example .env
```

3. Generate Prisma client
```bash
npm run prisma:generate
```

4. Run migrations
```bash
npm run prisma:migrate -- --name init
```

5. Seed admin user (optional)
```bash
npm run prisma:seed
```

6. Start dev server
```bash
npm run dev
```

## Prisma Configuration

- Datasource: PostgreSQL via `DATABASE_URL`
- ORM client: generated by `prisma generate`
- Initial model: `User` with `Role` enum (`ADMIN`, `STAFF`)
- Prisma config file: `prisma.config.ts`

See: `prisma/schema.prisma`

## Auth + RBAC Flow

- `POST /api/v1/auth/login` validates credentials, returns JWT.
- `authenticate` middleware verifies `Bearer <token>` and attaches `req.authUser`.
- `authorize(['ADMIN'])` middleware blocks non-admin users.

## Example Protected Routes

- Authenticated route: `GET /api/v1/users/me`
- Admin-only route: `GET /api/v1/users/admin`

Example:
```bash
curl -H "Authorization: Bearer <token>" http://localhost:4000/api/v1/users/me
curl -H "Authorization: Bearer <token>" http://localhost:4000/api/v1/users/admin
```
